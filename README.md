# My-Protfolio
Aspiring Data Analyst portfolio featuring real-world data projects and insights.

##Personal Portfolio Website

This is my personal portfolio website that showcases my skills, projects, resume, and professional information.
The website is designed to present my profile in a clean, structured, and interactive way.

 ##Project Overview

The Personal Portfolio Website is a responsive web application built using HTML, CSS, and JavaScript.
It helps visitors learn about me, view my work, and explore my skills and experience in an organized format.

 ##Features

✔ Responsive and modern design
✔ Home section with introduction
✔ About section with personal details
✔ Resume section with education, skills, and certifications
✔ Projects showcase section
✔ Contact section
✔ Smooth animations and hover effects
✔ Interactive functionality using JavaScript

 ##Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout design

JavaScript – Interactivity and dynamic behavior
